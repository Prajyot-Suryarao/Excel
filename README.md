# 📊 Zepto Sales Analysis Dashboard & Interactive Dashboard (Excel)

## 🔎 Project Overview
This project analyzes Zepto sales data to uncover business insights related to product performance, revenue generation, discounts, customer orders, and influencer marketing impact.
The analysis is performed using Excel Pivot Tables and an interactive Dashboard.

The dataset contains 300 sales records across multiple product categories and cities, helping identify sales trends and revenue-driving factors.

---

## 🎯 Business Objectives
- Analyze sales performance across products and categories.
- Evaluate revenue generation by city and product.
- Measure the impact of discounts on sales.
- Study the effectiveness of influencer marketing campaigns.
- Create an interactive dashboard for business decision-making.
